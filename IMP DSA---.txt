ğŸ“Œ2. str.Equals() Method
Behavior:
Also compares contents of two strings.
By default, case-sensitive.
Allows more control with options.
ğŸ‘‡ Code:EG:

{

    		{ "banana", 20 },

	};

	Console.WriteLine("The key related to the target value is " + foundkey) 
}
-FirstOrDefault() is a LINQ method that:
Returns the first element of a sequence (like an array, list, dictionary, etc.).
-What it does:
It searches for the first key-value pair in the dictionary (myMap) where the value matches the targetValue (20), and then returns the key of that pair.
