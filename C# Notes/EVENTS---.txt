Q1. Differences Between Delegates and Events
Feature						Delegate					Event
Definition			A reference type that holds method references			A wrapper around a delegate that restricts access

Usage				Used to call methods dynamically (callback mechanism)		Used for publish-subscribe communication

Invocation			Can be called (invoked) directly				Can only be invoked inside the declaring class

Encapsulation			No restrictions on invocation from other classes		Can only be subscribed (+=) or unsubscribed (-=) externally

Multiple Subscribers		Supports multiple method references (+=, -=)			Also supports multiple subscribers but cannot be invoked 												externally

Purpose				A function pointer mechanism for calling methods dynamically	Used to signal that an event has occurred

Default 			Storage	No automatic storage mechanism				Stores handlers using the event keyword



Q2)What is a Delegate?
A delegate is a type-safe function pointer that can refer to methods.
It allows methods to be assigned dynamically and invoked later.
EG :using System;

public delegate void MyDelegate(string message); // Declaring a delegate

class Program
{
    static void ShowMessage(string msg) => Console.WriteLine("Message: " + msg);

    static void Main()
    {
        MyDelegate del = ShowMessage; // Assigning a method to a delegate
        del("Hello, Delegates!");     // Calling the method via delegate
    }
}


Q3 . What is an Event?
An event is a special kind of delegate used in the publisher-subscriber model.
Only the declaring class can invoke the event, ensuring better encapsulation.

EG:
using System;

public class EventPublisher
{
    public event EventHandler MyEvent;  // Declare an event

    public void RaiseEvent()
    {
        Console.WriteLine("Raising an event...");
        MyEvent?.Invoke(this, EventArgs.Empty);  // Invoke event if there are subscribers
    }
}

class Program
{
    static void Main()
    {
        EventPublisher publisher = new EventPublisher();

        // Subscribe to the event
        publisher.MyEvent += (sender, e) => Console.WriteLine("Event received!");

        publisher.RaiseEvent(); // Raise the event
    }
}

EG 2 : for a button click
using System;
using System.Windows.Forms;

class Program
{
    static void Main()
    {
        Button button = new Button { Text = "Click Me" };
        button.Click += (sender, e) => MessageBox.Show("Button Clicked!");
        Application.Run(new Form { Controls = { button } });
    }
}
