ğŸ“Œ3.Custom HTML Helpers:
You can create custom HTML helpers by defining reusable methods to generate specific HTML markup. These helpers are typically used when you need custom functionality that the built-in helpers do not provide.
-Custom HTML Helpers are user-defined C# extension methods that generate reusable and custom HTML markup for your Razor views.
-When to Use
You need to generate a specific set of HTML repeatedly.
The built-in HTML helpers (Html.TextBoxFor, etc.) are not sufficient.
You want DRY (Don't Repeat Yourself) Razor views.

ğŸ”¹ Step 1: Create a Static Helper Class
Create a new static class (usually in a Helpers folder).
ğŸ“„ Helpers/CustomHtmlHelpers.cs
using Microsoft.AspNetCore.Html;
using Microsoft.AspNetCore.Mvc.Rendering;
using System;
using System.Linq.Expressions;

namespace YourApp.Helpers
{
    public static class CustomHtmlHelpers
    {
        // A simple Bootstrap styled TextBox helper
        public static IHtmlContent BootstrapTextBoxFor<TModel, TValue>(
            this IHtmlHelper<TModel> htmlHelper,
            Expression<Func<TModel, TValue>> expression,
            string placeholder = "")
        {
            return htmlHelper.TextBoxFor(expression, new
            {
                @class = "form-control",
                placeholder = placeholder
            });
        }
    }
}
